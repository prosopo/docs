title: "Spectra"
description: "Spectra-Unterstützung."
---

Das Procaptcha-Plugin unterstützt die [Forms-Funktion](https://wpspectra.com/docs/spectra-forms-block/) des [Spectra-Plugins](https://wordpress.org/plugins/ultimate-addons-for-gutenberg/).

Um Procaptcha zu jedem Spectra Gutenberg-Formular hinzuzufügen, folgen Sie diesen Schritten:

1. Wählen Sie den gewünschten Formular-Block im Gutenberg-Editor aus
2. Fügen Sie ein verstecktes Eingabefeld hinzu: Klicken Sie auf das Plus-Symbol und wählen Sie das 'hidden'-Feld
3. Verwenden Sie `prosopo_procaptcha` als Name und Wert des versteckten Feldes
4. Speichern Sie die Seite

Das war's! Jetzt können Sie die Seite öffnen (als nicht autorisierter Benutzer), um das Procaptcha-Widget an der Stelle des versteckten Feldes zu sehen.
