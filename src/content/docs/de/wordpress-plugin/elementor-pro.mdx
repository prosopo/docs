---
title: "Elementor Pro"
description: "Elementor Pro-Unterstützung."
---

Das Procaptcha-Plugin erweitert die folgenden [Elementor Pro](https://elementor.com/)-Widgets:

**1. Formular-Widget**

Um den Schutz für das [Formular-Widget](https://elementor.com/help/form-widget/) zu aktivieren:

1. Bearbeiten Sie die Zielseite mit dem gewünschten Formular-Widget.
2. Klicken Sie auf "Bearbeiten" beim Formular-Widget.
3. Klicken Sie, um ein neues Feld hinzuzufügen und wählen Sie `Prosopo Procaptcha` aus der Liste.
4. Speichern Sie die Seite.

**2. Login-Widget**

> Wichtig: Stellen Sie vor der Verwendung dieser Integration sicher, dass das Login-Formular in den Procaptcha-Einstellungen unter dem Tab `Core forms` aktiviert ist.

Um den Schutz für das [Login-Widget](https://elementor.com/help/login-widget-pro/) zu aktivieren:

1. Bearbeiten Sie die Zielseite mit dem gewünschten Login-Widget.
2. Klicken Sie auf "Bearbeiten" beim Login-Widget.
3. Aktivieren Sie im Abschnitt `Form Fields` das Kontrollkästchen `Prosopo Procaptcha`.
4. Speichern Sie die Seite.
