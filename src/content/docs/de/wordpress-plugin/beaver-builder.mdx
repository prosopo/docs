title: "Beaver Builder"
description: "Beaver Builder-Unterstützung."
---

Das Procaptcha-Plugin erweitert die folgenden [Beaver Builder](https://elementor.com/)-Formularelemente:

## 1. Kontaktformular

Um den Schutz für das [Kontaktformular](https://docs.wpbeaverbuilder.com/beaver-builder/layouts/modules/contact-form/) zu aktivieren:

1. Bearbeiten Sie die Zielseite mit dem gewünschten Formularelement.
2. Klicken Sie auf "Bearbeiten" beim Formularelement.
3. Wählen Sie im Abschnitt "General" die Option "Enabled" im Feld "Procaptcha protection".
4. Speichern Sie die Seite.

## 2. Abonnieren-Formular

Um den Schutz für das [Abonnieren-Formular](https://docs.wpbeaverbuilder.com/beaver-builder/2.9/layouts/modules/subscribe-form/) zu aktivieren:

1. Bearbeiten Sie die Zielseite mit dem gewünschten Formularelement.
2. Klicken Sie auf "Bearbeiten" beim Formularelement.
3. Wählen Sie im Abschnitt "General" die Option "Enabled" im Feld "Procaptcha protection".
4. Speichern Sie die Seite.

## 3. Login-Formular

Im Gegensatz zu den oben genannten Formularen hat das [Login-Formular](https://docs.wpbeaverbuilder.com/beaver-builder/2.9/layouts/modules/login-form) keine separate Procaptcha-Einstellung.

Stattdessen wird es über die generischen Procaptcha-Einstellungen gesteuert:

1. Besuchen Sie die Procaptcha-Plugin-Einstellungen
2. Wechseln Sie zum Tab "Core forms"
3. Aktivieren Sie den Login-Formular-Schutz
4. Speichern Sie die Einstellungen
