title: "JetPack"
description: "JetPack Forms-Unterstützung."
---

Das Procaptcha-Plugin unterstützt die [Forms-Funktion](https://jetpack.com/forms/) des [JetPack-Plugins](https://wordpress.org/plugins/jetpack/).

Um Procaptcha zu jedem JetPack Gutenberg-Formular hinzuzufügen, folgen Sie diesen Schritten:

1. Fügen Sie den [Group-Block](https://wordpress.org/documentation/article/group-block/) innerhalb des Zielformular-Blocks hinzu
2. Fügen Sie den [Shortcode-Block](https://wordpress.org/documentation/article/shortcode-block/) innerhalb des hinzugefügten Group-Blocks hinzu
3. Fügen Sie `[prosopo_procaptcha]` innerhalb des hinzugefügten Shortcode-Blocks ein
4. Speichern Sie die Seite

Das war's! Jetzt können Sie die Seite öffnen (als nicht autorisierter Benutzer), um das Procaptcha-Widget innerhalb des Formulars zu sehen.
