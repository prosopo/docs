---
title: "Memberpress"
description: "Memberpress-Unterstützung."
---

Das Procaptcha-Plugin erweitert die folgenden [Memberpress](https://memberpress.com/)-Formulare:

## 1. Mitgliedschaftsregistrierung

Um den Schutz für das [Mitgliedschaftsregistrierungsformular](https://memberpress.com/docs/creating-memberships/) zu aktivieren:

1. Bearbeiten Sie die Zielmitgliedschaft.
2. Schalten Sie in den Mitgliedschaftsoptionen unter dem Abschnitt Registrierung das Kontrollkästchen "Procaptcha form protection" ein.
3. Speichern Sie die Mitgliedschaft.

## 2. Login-Formular

Der Formularschutz für das Login-Formular wird über die generischen Procaptcha-Einstellungen gesteuert:

1. Besuchen Sie die Procaptcha-Plugin-Einstellungen
2. Wechseln Sie zum Tab "Core forms"
3. Aktivieren Sie den Login-Formular-Schutz
4. Speichern Sie die Einstellungen

## 3. Passwort-Zurücksetzen-Formular

Der Formularschutz für das Passwort-Zurücksetzen-Formular wird über die generischen Procaptcha-Einstellungen gesteuert:

1. Besuchen Sie die Procaptcha-Plugin-Einstellungen
2. Wechseln Sie zum Tab "Core forms"
3. Aktivieren Sie den Passwort-vergessen-Formular-Schutz
4. Speichern Sie die Einstellungen
