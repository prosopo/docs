Procaptcha bietet eine [offizielle React-Komponente](https://www.npmjs.com/package/@prosopo/react-procaptcha-wrapper) für die einfache Integration von Prosopo Procaptcha in jedes React-Projekt.
Die Integrationskomponente unterstützt alle [Procaptcha Render-Optionen](/de/basics/client-side-rendering/#procaptcha-optionen).

## 1. Installation

`npm install @prosopo/react-procaptcha-wrapper`

## 2. Verwendungsbeispiel

### 2.1) Basis-Setup

```typescript jsx
import {ProcaptchaComponent} from "@prosopo/react-procaptcha-wrapper";

// ...

<ProcaptchaComponent siteKey={"my-site-key"}/>;
```

### 2.2) Erweiterte Verwendung

```typescript jsx
// ...

<ProcaptchaComponent
    siteKey={siteKey}
    language={"en"}
    callback={(token: string): void => {
        console.log("verified", token);
    }}
    htmlAttributes={{
        className: "my-app__procaptcha",
        style: {
            maxWidth: "600px",
        },
    }}
/>
```
