Procaptcha fornece um component Svelte oficial para fácil integração do Prosopo Procaptcha em qualquer projeto Svelte.
O component de integração suporta todas as [opções de renderização do Procaptcha](/pt-br/basics/client-side-rendering/#procaptcha-options).

## 1. Instalação

`npm install @prosopo/svelte-procaptcha-wrapper`

## 2. Exemplo de uso

### 2.1) Configuração básica

```svelte
<script lang="ts">
import {ProcaptchaComponent} from "@prosopo/svelte-procaptcha-wrapper";
</script>

<ProcaptchaComponent siteKey={"my-site-key"}/>;
```

### 2.2) Uso avançado

```svelte
<script lang="ts">
import {ProcaptchaComponent} from "@prosopo/svelte-procaptcha-wrapper";

const siteKey = "my-site-key";
const handleVerification = (token: string): void => {
    console.log('verified', token);
};
</script>

<ProcaptchaComponent siteKey="{siteKey}"
language="en"
callback={handleVerification}
htmlAttributes={{class: "my-app__procaptcha"}} />
```
