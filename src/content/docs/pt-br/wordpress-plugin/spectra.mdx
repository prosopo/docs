O plugin Procaptcha suporta o [recurso de formulários](https://wpspectra.com/docs/spectra-forms-block/) do [plugin Spectra](https://wordpress.org/plugins/ultimate-addons-for-gutenberg/).

Para adicionar o Procaptcha a qualquer formulário Gutenberg do Spectra, siga estas etapas:

1. Selecione o bloco de Formulário desejado no editor Gutenberg
2. Adicione um campo de entrada oculto: clique no ícone de adição e opte pelo campo 'hidden'
3. Use `prosopo_procaptcha` como nome e valor do campo oculto
4. Salve a página

É isso! Agora você pode abrir a página (como um usuário não autorizado) para ver o widget Procaptcha no lugar do campo oculto.
