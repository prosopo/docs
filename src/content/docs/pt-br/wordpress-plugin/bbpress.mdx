O plugin Procaptcha suporta os seguintes formulários do [bbPress](https://wordpress.org/plugins/bbpress/):

**1. Formulários de fórum**

O plugin Procaptcha adiciona a configuração `Prosopo Procaptcha` a todos os fóruns.

Visite a configuração do fórum de destino para ativar a proteção para formulários de criação de tópico e resposta.

**2. Formulários de conta**

O bbPress vem com uma [série de shortcodes](https://codex.bbpress.org/features/shortcodes/), que permitem criar formulários de conta personalizados: Registro, Login e Recuperar Senha.
Para ativar a proteção deles, visite as configurações do Procaptcha, abra a aba `Core Forms` e ative os seletores relacionados.
