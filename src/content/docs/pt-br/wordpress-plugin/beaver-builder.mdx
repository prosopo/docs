O plugin Procaptcha estende os seguintes elementos de formulário do [Beaver Builder](https://elementor.com/):

## 1. Formulário de contato

Para ativar a proteção para o [formulário de contato](https://docs.wpbeaverbuilder.com/beaver-builder/layouts/modules/contact-form/):

1. Edite a página de destino com o elemento de formulário desejado.
2. Clique em editar no elemento de formulário.
3. Na seção Geral, opte pela opção "Habilitado" no campo "Proteção Procaptcha".
4. Salve a página.

## 2. Formulário de inscrição

Para ativar a proteção para o [formulário de inscrição](https://docs.wpbeaverbuilder.com/beaver-builder/2.9/layouts/modules/subscribe-form/):

1. Edite a página de destino com o elemento de formulário desejado.
2. Clique em editar no elemento de formulário.
3. Na seção Geral, opte pela opção "Habilitado" no campo "Proteção Procaptcha".
4. Salve a página.

## 3. Formulário de login

Diferentemente dos formulários acima, o [formulário de login](https://docs.wpbeaverbuilder.com/beaver-builder/2.9/layouts/modules/login-form) não possui uma configuração Procaptcha separada.

Em vez disso, ele é controlado através das configurações genéricas do Procaptcha:

1. Visite as configurações do plugin Procaptcha
2. Mude para a aba "Core forms"
3. Ative a proteção do formulário de login
4. Salve as configurações
