O plugin Procaptcha estende o [Ninja Forms](https://wordpress.org/plugins/ninja-forms/) registrando o campo `Prosopo Procaptcha`, que você pode facilmente adicionar aos seus formulários para proteção Procaptcha.

Para ativar isso, abra o formulário de destino e insira o campo `Prosopo Procaptcha` em qualquer lugar do formulário, acima do botão `Submit`.
