O plugin Procaptcha estende os seguintes formulários do [Memberpress](https://memberpress.com/):

## 1. Registro de associação

Para ativar a proteção para o [formulário de registro de associação](https://memberpress.com/docs/creating-memberships/):

1. Edite a associação de destino.
2. Nas opções de Associação, na seção Registro, ative a checkbox "Proteção de formulário Procaptcha".
3. Salve a associação.

## 2. Formulário de login

A proteção do formulário de login é controlada através das configurações genéricas do Procaptcha:

1. Visite as configurações do plugin Procaptcha
2. Mude para a aba "Core forms"
3. Ative a proteção do formulário de login
4. Salve as configurações

## 3. Formulário de redefinição de senha

A proteção do formulário de redefinição de senha é controlada através das configurações genéricas do Procaptcha:

1. Visite as configurações do plugin Procaptcha
2. Mude para a aba "Core forms"
3. Ative a proteção do formulário de recuperação de senha
4. Salve as configurações
