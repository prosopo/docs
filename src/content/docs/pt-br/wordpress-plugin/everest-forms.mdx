O plugin Procaptcha estende o [Everest Forms](https://wordpress.org/plugins/everest-forms/) registrando o campo `Prosopo Procaptcha`, que você pode facilmente adicionar aos seus formulários para proteção Procaptcha.

Para ativar isso, abra o formulário de destino e insira o campo `Prosopo Procaptcha` em qualquer lugar do formulário, acima do botão `Submit`.
