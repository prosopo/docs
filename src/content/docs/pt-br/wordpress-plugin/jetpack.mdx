O plugin Procaptcha suporta o [recurso de formulários](https://jetpack.com/forms/) do [plugin JetPack](https://wordpress.org/plugins/jetpack/).

Para adicionar o Procaptcha a qualquer formulário Gutenberg do JetPack, siga estas etapas:

1. Adicione o [bloco de Grupo](https://wordpress.org/documentation/article/group-block/) dentro do bloco de formulário de destino
2. Adicione o [bloco de Shortcode](https://wordpress.org/documentation/article/shortcode-block/) dentro do bloco de Grupo adicionado
3. Insira o `[prosopo_procaptcha]` dentro do bloco de Shortcode adicionado
4. Salve a página

É isso! Agora você pode abrir a página (como um usuário não autorizado) para ver o widget Procaptcha dentro do formulário.
