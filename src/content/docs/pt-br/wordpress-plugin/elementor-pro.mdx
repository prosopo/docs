O plugin Procaptcha estende os seguintes widgets do [Elementor Pro](https://elementor.com/):

**1. Widget de formulário**

Para ativar a proteção para o [widget de formulário](https://elementor.com/help/form-widget/):

1. Edite a página de destino com o widget de formulário desejado.
2. Clique em editar no widget de formulário.
3. Clique para adicionar um novo campo e selecione `Prosopo Procaptcha` da lista.
4. Salve a página.

**2. Widget de login**

> Importante: Antes de usar esta integração, certifique-se de que o Formulário de Login está habilitado nas configurações do Procaptcha na aba `Core forms`.

Para ativar a proteção para o [widget de login](https://elementor.com/help/login-widget-pro/):

1. Edite a página de destino com o widget de login desejado.
2. Clique em editar no widget de login.
3. Na seção `Form Fields`, habilite a checkbox `Prosopo Procaptcha`.
4. Salve a página.
