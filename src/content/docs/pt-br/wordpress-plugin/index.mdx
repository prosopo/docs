O Procaptcha possui um [plugin oficial para WordPress](https://wordpress.org/plugins/prosopo-procaptcha/), possibilitando integração perfeita com seu site WordPress.

## 1. Instalação

1. Visite o painel de administração do WordPress e vá para **Plugins** > **Adicionar novo**, depois pesquise por **Procaptcha** e clique em **Instalar agora**.
2. Quando a instalação estiver completa, clique em **Ativar**.
3. Navegue até as configurações do plugin (**Configurações > Procaptcha**) e insira sua sitekey e secret key, que você pode obter no [portal do usuário Prosopo](https://portal.prosopo.io/).
4. Salve as configurações

Se as credenciais estiverem corretas, você verá o Procaptcha exibido no lado direito da página de configurações.

## 2. Plugins de Formulário Suportados

O Procaptcha integra-se com muitos plugins de formulário populares.

Após a instalação, visite as configurações do plugin e abra a aba "Active Integrations" para ver uma lista de plugins compatíveis e links para os guias de integração.
Alternativamente, navegue pelas páginas no menu esquerdo desta documentação.

Não encontrou seu plugin? Avise-nos através do [fórum de suporte](https://wordpress.org/support/plugin/prosopo-procaptcha/), e consideraremos adicioná-lo!

## 3. Formulários Principais do WordPress Suportados

O Procaptcha suporta formulários principais do WordPress nativamente:

1. Login
2. Registro
3. Recuperar Senha
4. Comentários
5. Proteção de senha de Post/Página

Visite as configurações do plugin para ativar a proteção para estes formulários.

## 7. FAQ

**O Procaptcha é visível apenas para usuários não autorizados?**

Sim, por padrão, o Procaptcha é mostrado apenas para usuários não autorizados. Você pode alterar isso na seção `behavior` das configurações do plugin, ou usando os hooks do plugin.

Nota: No entanto, alguns plugins, como Contact Form 7, não enviam dados do usuário durante o envio, portanto o Procaptcha é exibido para todos os usuários nesses casos.
