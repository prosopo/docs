O plugin Procaptcha estende o [User Registration & Membership](https://wordpress.org/plugins/user-registration/):

1. Registrando o campo `Prosopo Procaptcha`, que você pode facilmente adicionar aos seus formulários de registro para proteção Procaptcha.

Para ativar isso, abra o formulário de destino e insira o campo `Prosopo Procaptcha` em qualquer lugar do formulário, acima do botão `Submit`.

2. Estendendo os formulários personalizados de login e redefinição de senha.

O plugin User Registration permite não apenas criar formulários de inscrição, mas também elaborar formulários de Login e Redefinição de Senha.
Para ativar a proteção deles, visite as configurações do Procaptcha, abra a aba `Core Forms` e ative os seletores relacionados.
