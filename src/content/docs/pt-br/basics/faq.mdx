### O que significa `Site Key not Registered`?
Você pode encontrar este erro quando configurar o widget Procaptcha no seu site. Este erro ocorre quando a chave do site
que você está usando não está registrada no sistema Prosopo. Para resolver este erro, clique no link de Ativação no e-mail
ou <a href="mailto:support@prosopo.io">entre em contato com a equipe de suporte Prosopo</a>.

### O que significa `Invalid Site Key`?
Este erro ocorre quando a chave do site que você está usando não é válida. Para resolver este erro, certifique-se de que está usando a
chave do site correta, que você pode encontrar fazendo login no [portal Prosopo](https://portal.prosopo.io).

### O que significa `Unauthorized origin URL`?
Este erro ocorre quando o domínio da chave do site não corresponde ao domínio do site onde o widget Procaptcha
está sendo usado. Para resolver este erro, certifique-se de que o domínio da chave do site corresponde ao domínio do site
fazendo login no [portal Prosopo](https://portal.prosopo.io) e atualizando o domínio.

### O que significa `Incorrect CAPTCHA type`?
Este erro ocorre quando o [tipo de captcha](/pt-br/basics/captcha-types) não está definido corretamente. O `captchaType` padrão é
`frictionless`. Se você deseja usar `pow` ou `image`, deve definir seu `captchaType` na configuração do widget para `pow`
ou `image`. Você também deve atualizar suas [configurações do portal](https://portal.prosopo.io) para corresponder.

### O que significa `Session not found`?
Este erro ocorre quando a sessão não é encontrada. Isso pode acontecer se a sessão expirou ou se o ID da sessão
não é válido. Para resolver este erro, aguarde o widget recarregar (3s) ou atualize a página. Se o erro persistir,
tente limpar o cache e os cookies do seu navegador. Se o erro ainda persistir, [entre em contato com a equipe de suporte Prosopo](https://discord.com/invite/3nMYAHecZT).

### O que significa `Error parsing captcha`?
Este erro ocorrerá se dados incorretos foram enviados com a resposta do captcha. Para resolver este erro, certifique-se de que os
dados enviados com a resposta do captcha estão corretos.
