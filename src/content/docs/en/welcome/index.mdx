Prosopo Procaptcha is an [open-source](https://github.com/prosopo/captcha), drop-in replacement for reCAPTCHA, hCaptcha, and Cloudflare Turnstile that protects user privacy and collects minimal data.

<Image src={ProcaptchaGIF} alt={'prosopo procaptcha pow captcha challenge'} />

[Sign up for free](https://prosopo.io/register) and get your sitekey today. You need a sitekey to use
this library.

## Get Started

Once you have your site key and secret key, follow our guide on [how to get started](/en/basics/) with Procaptcha.

## Demos

### HTML Demo

You can view an end-to-end example of how to implement Procaptcha in a simple HTML page in
our [client-bundle-example](/en/demos/client-example-bundle/)
and [client-example-server](/en/demos/client-example-server/) packages. Details
of how to run the examples are in the documentation at the previous links.
