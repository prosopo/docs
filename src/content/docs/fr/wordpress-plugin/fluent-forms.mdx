---
title: 'Fluent Forms'
description: 'Support de Fluent Forms.'
---

Le plugin Procaptcha étend [Fluent Forms](https://wordpress.org/plugins/fluentform/) en enregistrant le champ `Prosopo Procaptcha`, que vous pouvez facilement ajouter à vos formulaires pour la protection Procaptcha.

Pour l'activer, ouvrez le formulaire cible et insérez le champ `Prosopo Procaptcha` n'importe où dans le formulaire, au-dessus du bouton `Soumettre`.
