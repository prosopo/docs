---
title: 'WPForms'
description: 'Support de WPForms.'
---

Le plugin Procaptcha étend [WPForms](https://wordpress.org/plugins/wpforms-lite/) en enregistrant le champ `Prosopo Procaptcha`, que vous pouvez facilement ajouter à vos formulaires pour la protection Procaptcha.

Pour l'activer, ouvrez le formulaire cible et insérez le champ `Prosopo Procaptcha` n'importe où dans le formulaire, au-dessus du bouton `Soumettre`.
