Procaptcha dispose d'un [plugin WordPress officiel](https://wordpress.org/plugins/prosopo-procaptcha/), permettant une intégration transparente avec votre site WordPress.

## 1. Installation

1. Visitez votre panneau d'administration WordPress, et allez dans **Extensions** > **Ajouter**, puis recherchez **Procaptcha** et cliquez sur **Installer maintenant**.
2. Une fois l'installation terminée, cliquez sur **Activer**.
3. Accédez aux paramètres du plugin (**Réglages > Procaptcha**) et entrez votre clé de site et votre clé secrète, que vous pouvez obtenir depuis le [portail utilisateur Prosopo](https://portal.prosopo.io/).
4. Enregistrez les paramètres

Si les identifiants sont corrects, vous verrez le Procaptcha affiché sur le côté droit de la page des paramètres.

## 2. Plugins de formulaire pris en charge

Procaptcha s'intègre avec de nombreux plugins de formulaire populaires.

Après l'installation, visitez les paramètres du plugin et ouvrez l'onglet "Intégrations actives" pour voir une liste de plugins compatibles et des liens vers les guides d'intégration.
Alternativement, parcourez les pages dans le menu de gauche de cette documentation.

Vous ne trouvez pas votre plugin ? Faites-le nous savoir via le [forum de support](https://wordpress.org/support/plugin/prosopo-procaptcha/), et nous envisagerons de l'ajouter !

## 3. Formulaires WordPress de base pris en charge

Procaptcha prend en charge les formulaires de base de WordPress prêts à l'emploi :

1. Connexion
2. Inscription
3. Mot de passe perdu
4. Commentaires
5. Protection par mot de passe des articles/pages

Visitez les paramètres du plugin pour activer la protection de ces formulaires.

## 7. FAQ

**Procaptcha est-il visible uniquement pour les utilisateurs non autorisés ?**

Oui, par défaut, Procaptcha n'est affiché qu'aux utilisateurs non autorisés. Vous pouvez modifier cela dans la section `comportement` des paramètres du plugin, ou en utilisant les hooks du plugin.

Note : Cependant, certains plugins, tels que Contact Form 7, n'envoient pas de données utilisateur lors de la soumission, donc Procaptcha est affiché à tous les utilisateurs dans ces cas.
