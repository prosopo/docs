Le plugin Procaptcha étend les formulaires [Memberpress](https://memberpress.com/) suivants :

## 1. Inscription à l'adhésion

Pour activer la protection du [formulaire d'inscription à l'adhésion](https://memberpress.com/docs/creating-memberships/) :

1. Modifiez l'adhésion cible.
2. Dans les options d'adhésion, sous la section Inscription, activez la case à cocher "Protection du formulaire Procaptcha".
3. Enregistrez l'adhésion.

## 2. Formulaire de connexion

La protection du formulaire de connexion est contrôlée via les paramètres génériques de Procaptcha :

1. Visitez les paramètres du plugin Procaptcha
2. Passez à l'onglet "Formulaires de base"
3. Activez la protection du formulaire de connexion
4. Enregistrez les paramètres

## 3. Formulaire de réinitialisation du mot de passe

La protection du formulaire de réinitialisation du mot de passe est contrôlée via les paramètres génériques de Procaptcha :

1. Visitez les paramètres du plugin Procaptcha
2. Passez à l'onglet "Formulaires de base"
3. Activez la protection du formulaire de mot de passe perdu
4. Enregistrez les paramètres
