---

Le plugin Procaptcha étend les widgets [Elementor Pro](https://elementor.com/) suivants :

**1. Widget de formulaire**

Pour activer la protection du [widget de formulaire](https://elementor.com/help/form-widget/) :

1. Modifiez la page cible avec le widget de formulaire souhaité.
2. Cliquez sur modifier sur le widget de formulaire.
3. Cliquez pour ajouter un nouveau champ et sélectionnez `Prosopo Procaptcha` dans la liste.
4. Enregistrez la page.

**2. Widget de connexion**

> Important : Avant d'utiliser cette intégration, assurez-vous que le formulaire de connexion est activé dans les paramètres Procaptcha sous l'onglet `Formulaires de base`.

Pour activer la protection du [widget de connexion](https://elementor.com/help/login-widget-pro/) :

1. Modifiez la page cible avec le widget de connexion souhaité.
2. Cliquez sur modifier sur le widget de connexion.
3. Dans la section `Champs de formulaire`, activez la case à cocher `Prosopo Procaptcha`.
4. Enregistrez la page.
