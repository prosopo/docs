---
title: 'Ninja Forms'
description: 'Support de Ninja Forms.'
---

Le plugin Procaptcha étend [Ninja Forms](https://wordpress.org/plugins/ninja-forms/) en enregistrant le champ `Prosopo Procaptcha`, que vous pouvez facilement ajouter à vos formulaires pour la protection Procaptcha.

Pour l'activer, ouvrez le formulaire cible et insérez le champ `Prosopo Procaptcha` n'importe où dans le formulaire, au-dessus du bouton `Soumettre`.
