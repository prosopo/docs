Le plugin Procaptcha étend les éléments de formulaire [Beaver Builder](https://elementor.com/) suivants :

## 1. Formulaire de contact

Pour activer la protection du [formulaire de contact](https://docs.wpbeaverbuilder.com/beaver-builder/layouts/modules/contact-form/) :

1. Modifiez la page cible avec l'élément de formulaire souhaité.
2. Cliquez sur modifier sur l'élément de formulaire.
3. Dans la section Général, optez pour l'option "Activé" dans le champ "Protection Procaptcha".
4. Enregistrez la page.

## 2. Formulaire d'abonnement

Pour activer la protection du [formulaire d'abonnement](https://docs.wpbeaverbuilder.com/beaver-builder/2.9/layouts/modules/subscribe-form/) :

1. Modifiez la page cible avec l'élément de formulaire souhaité.
2. Cliquez sur modifier sur l'élément de formulaire.
3. Dans la section Général, optez pour l'option "Activé" dans le champ "Protection Procaptcha".
4. Enregistrez la page.

## 3. Formulaire de connexion

Contrairement aux formulaires ci-dessus, le [formulaire de connexion](https://docs.wpbeaverbuilder.com/beaver-builder/2.9/layouts/modules/login-form) n'a pas de paramètre Procaptcha séparé.

Au lieu de cela, il est contrôlé via les paramètres génériques de Procaptcha :

1. Visitez les paramètres du plugin Procaptcha
2. Passez à l'onglet "Formulaires de base"
3. Activez la protection du formulaire de connexion
4. Enregistrez les paramètres
