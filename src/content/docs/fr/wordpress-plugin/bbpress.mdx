Le plugin Procaptcha prend en charge les formulaires [bbPress](https://wordpress.org/plugins/bbpress/) suivants :

**1. Formulaires de forum**

Le plugin Procaptcha ajoute le paramètre `Prosopo Procaptcha` à chaque forum.

Visitez le paramètre du forum cible pour activer la protection pour la création de sujets et les formulaires de réponse.

**2. Formulaires de compte**

bbPress est livré avec une [série de shortcodes](https://codex.bbpress.org/features/shortcodes/), qui permettent de créer des formulaires de compte personnalisés : Inscription, Connexion et Mot de passe perdu.
Pour activer leur protection, visitez les paramètres Procaptcha, ouvrez l'onglet `Formulaires de base` et activez les commutateurs correspondants.
