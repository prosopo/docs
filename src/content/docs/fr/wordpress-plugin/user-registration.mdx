Le plugin Procaptcha étend [User Registration & Membership](https://wordpress.org/plugins/user-registration/) :

1. En enregistrant le champ `Prosopo Procaptcha`, que vous pouvez facilement ajouter à vos formulaires d'inscription pour la protection Procaptcha.

Pour l'activer, ouvrez le formulaire cible et insérez le champ `Prosopo Procaptcha` n'importe où dans le formulaire, au-dessus du bouton `Soumettre`.

2. En étendant les formulaires personnalisés de connexion et de réinitialisation de mot de passe.

Le plugin User Registration permet non seulement de créer des formulaires d'inscription, mais aussi de créer des formulaires de connexion et de réinitialisation de mot de passe.
Pour activer leur protection, visitez les paramètres Procaptcha, ouvrez l'onglet `Formulaires de base` et activez les commutateurs correspondants.
