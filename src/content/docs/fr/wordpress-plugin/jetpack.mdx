Le plugin Procaptcha prend en charge la [fonctionnalité de formulaires](https://jetpack.com/forms/) du [plugin JetPack](https://wordpress.org/plugins/jetpack/).

Pour ajouter Procaptcha à n'importe quel formulaire Gutenberg JetPack, suivez ces étapes :

1. Ajoutez le [bloc Groupe](https://wordpress.org/documentation/article/group-block/) à l'intérieur du bloc de formulaire cible
2. Ajoutez le [bloc Shortcode](https://wordpress.org/documentation/article/shortcode-block/) à l'intérieur du bloc Groupe ajouté
3. Insérez le `[prosopo_procaptcha]` à l'intérieur du bloc Shortcode ajouté
4. Enregistrez la page

C'est tout ! Maintenant vous pouvez ouvrir la page (en tant qu'utilisateur non autorisé) pour voir le widget Procaptcha à l'intérieur du formulaire.
