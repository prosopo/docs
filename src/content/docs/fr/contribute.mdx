Nous accueillons les contributions de toute taille et les contributeurs de tous niveaux de compétence. En tant que projet open-source, nous croyons en la reconnaissance de nos contributeurs. Nous sommes heureux d'aider avec des conseils sur les PR, la rédaction technique et la transformation de toute idée de fonctionnalité en réalité.

Vous voulez vous impliquer encore plus ? Consultez notre [document de gouvernance](https://github.com/prosopo/.github/blob/main/GOVERNANCE.md) pour des descriptions détaillées des différents rôles, des processus de nomination des mainteneurs, des processus de révision de code et de l'application du code de conduite.

## Façons de contribuer

### Dépôts de projets

Il existe de nombreuses façons de contribuer au projet Prosopo ! Chaque dépôt Prosopo a un README avec un lien vers un fichier `CONTRIBUTING.md` à la racine du projet.

Visitez [le profil GitHub de Prosopo](https://github.com/prosopo) pour trouver les dépôts pour :

- La [base de code principale de Prosopo Procaptcha](https://github.com/prosopo/captcha).

- [Prosopo Docs](https://github.com/prosopo/docs), un site web Prosopo entier ! Contribuez non seulement du contenu écrit, mais aussi du code Prosopo traitant des préoccupations d'accessibilité, CSS, UI et UX. Nous mettons également notre documentation à disposition en plusieurs langues, nous avons donc besoin d'aide pour traduire l'ensemble du site.

### Types de contributions

En plus de contribuer votre propre code ou contenu, vous pouvez également apporter une énorme contribution en vous impliquant en laissant des commentaires de révision sur les PR, en ajoutant des idées dans les issues et discussions GitHub existantes, et en participant à nos tâches de maintenance des issues "épinglées" !

Chaque PR, en particulier les PR de traduction, a besoin de réviseurs ! Réviser les PR et laisser des commentaires, des suggestions ou un "LGTM!" d'approbation ("Looks Good To Me!") est un excellent moyen de commencer dans n'importe quel dépôt et d'en apprendre davantage sur Prosopo.

Nous avons également une communauté [Discord](https://discord.gg/3nMYAHecZT) active ! Nous valorisons les contributions de ceux qui accueillent de nouveaux membres et répondent aux questions de support.

## Contribuer à la documentation

Veuillez soumettre une demande de PR dans [Prosopo Docs](https://github.com/prosopo/docs).
