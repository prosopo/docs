### Que signifie `Site Key not Registered` ?
Vous pouvez rencontrer cette erreur lorsque vous configurez le widget Procaptcha sur votre site web. Cette erreur se produit lorsque la clé de site que vous utilisez n'est pas enregistrée dans le système Prosopo. Pour résoudre cette erreur, cliquez sur le lien d'activation dans l'e-mail ou <a href="mailto:support@prosopo.io">contactez l'équipe de support Prosopo</a>.

### Que signifie `Invalid Site Key` ?
Cette erreur se produit lorsque la clé de site que vous utilisez n'est pas valide. Pour résoudre cette erreur, assurez-vous que vous utilisez la bonne clé de site, que vous pouvez trouver en vous connectant au [portail Prosopo](https://portal.prosopo.io).

### Que signifie `Unauthorized origin URL` ?
Cette erreur se produit lorsque le domaine de la clé de site ne correspond pas au domaine du site web où le widget Procaptcha est utilisé. Pour résoudre cette erreur, assurez-vous que le domaine de la clé de site correspond au domaine du site web en vous connectant au [portail Prosopo](https://portal.prosopo.io) et en mettant à jour le domaine.

### Que signifie `Incorrect CAPTCHA type` ?
Cette erreur se produit lorsque le [type de captcha](/fr/basics/captcha-types) n'est pas défini correctement. Le `captchaType` par défaut est `frictionless`. Si vous souhaitez utiliser `pow` ou `image`, vous devez définir votre `captchaType` dans la configuration du widget à `pow` ou `image`. Vous devez également mettre à jour vos [paramètres du portail](https://portal.prosopo.io) pour qu'ils correspondent.

### Que signifie `Session not found` ?
Cette erreur se produit lorsque la session n'est pas trouvée. Cela peut se produire si la session a expiré ou si l'ID de session n'est pas valide. Pour résoudre cette erreur, attendez que le widget se recharge (3s) ou actualisez la page. Si l'erreur persiste, essayez de vider le cache et les cookies de votre navigateur. Si l'erreur persiste toujours, veuillez [contacter l'équipe de support Prosopo](https://discord.com/invite/3nMYAHecZT).

### Que signifie `Error parsing captcha` ?
Cette erreur se produira si des données incorrectes ont été envoyées avec la réponse du captcha. Pour résoudre cette erreur, assurez-vous que les données envoyées avec la réponse du captcha sont correctes.
