Prosopo Procaptcha est un remplacement direct [open-source](https://github.com/prosopo/captcha) de reCAPTCHA, hCaptcha et Cloudflare Turnstile qui protège la vie privée des utilisateurs et collecte un minimum de données.

<Image src={ProcaptchaGIF} alt={'défi captcha pow prosopo procaptcha'} />

[Inscrivez-vous gratuitement](https://prosopo.io/register) et obtenez votre sitekey dès aujourd'hui. Vous avez besoin d'un sitekey pour utiliser cette bibliothèque.

## Commencer

Une fois que vous avez votre clé de site et votre clé secrète, suivez notre guide sur [comment commencer](/fr/basics/) avec Procaptcha.

## Démos

### Démo HTML

Vous pouvez voir un exemple de bout en bout de la façon d'implémenter Procaptcha dans une simple page HTML dans nos packages [client-bundle-example](/fr/demos/client-example-bundle/) et [client-example-server](/fr/demos/client-example-server/). Les détails sur la façon d'exécuter les exemples se trouvent dans la documentation aux liens précédents.
