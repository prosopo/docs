Il plugin Procaptcha estende i seguenti moduli [Memberpress](https://memberpress.com/):

## 1. Registrazione membership

Per abilitare la protezione per il [modulo di registrazione membership](https://memberpress.com/docs/creating-memberships/):

1. Modifichi la membership target.
2. Nelle opzioni Membership, nella sezione Registration, attivi la casella di controllo "Procaptcha form protection".
3. Salvi la membership.

## 2. Modulo di login

La protezione del modulo per il modulo di login è controllata tramite le impostazioni generiche di Procaptcha:

1. Visiti le impostazioni del plugin Procaptcha
2. Passi alla scheda "Core forms"
3. Abiliti la protezione del modulo di login
4. Salvi le impostazioni

## 3. Modulo di reset password

La protezione del modulo per il modulo di reset password è controllata tramite le impostazioni generiche di Procaptcha:

1. Visiti le impostazioni del plugin Procaptcha
2. Passi alla scheda "Core forms"
3. Abiliti la protezione del modulo password dimenticata
4. Salvi le impostazioni
