Procaptcha ha un [plugin ufficiale WordPress](https://wordpress.org/plugins/prosopo-procaptcha/), che consente un'integrazione perfetta con il suo sito WordPress.

## 1. Installazione

1. Visiti il pannello di amministrazione di WordPress e vada su **Plugin** > **Aggiungi nuovo**, quindi cerchi **Procaptcha** e clicchi su **Installa ora**.
2. Una volta completata l'installazione, clicchi su **Attiva**.
3. Navighi alle impostazioni del plugin (**Impostazioni > Procaptcha**) e inserisca la sua site key e secret key, che può ottenere dal [portale utente Prosopo](https://portal.prosopo.io/).
4. Salvi le impostazioni

Se le credenziali sono corrette, vedrà Procaptcha visualizzato sul lato destro della pagina delle impostazioni.

## 2. Plugin di moduli supportati

Procaptcha si integra con molti plugin di moduli popolari.

Dopo l'installazione, visiti le impostazioni del plugin e apra la scheda "Active Integrations" per vedere un elenco di plugin compatibili e link alle guide di integrazione.
In alternativa, sfoglia le pagine nel menu di sinistra di questa documentazione.

Non riesce a trovare il suo plugin? Ce lo faccia sapere tramite il [forum di supporto](https://wordpress.org/support/plugin/prosopo-procaptcha/), e valuteremo di aggiungerlo!

## 3. Moduli core WordPress supportati

Procaptcha supporta i moduli core di WordPress out-of-the-box:

1. Login
2. Registrazione
3. Password dimenticata
4. Commenti
5. Protezione password Post/Pagina

Visiti le impostazioni del plugin per abilitare la protezione per questi moduli.

## 7. FAQ

**Procaptcha è visibile solo agli utenti non autorizzati?**

Sì, per impostazione predefinita, Procaptcha è mostrato solo agli utenti non autorizzati. Può modificare questo nella sezione `behavior` delle impostazioni del plugin, o utilizzando gli hook del plugin.

Nota: Tuttavia, alcuni plugin, come Contact Form 7, non inviano i dati utente durante l'invio, quindi Procaptcha viene visualizzato a tutti gli utenti in questi casi.
