---

Il plugin Procaptcha estende i seguenti widget [Elementor Pro](https://elementor.com/):

**1. Widget modulo**

Per abilitare la protezione per il [widget modulo](https://elementor.com/help/form-widget/):

1. Modifichi la pagina target con il widget modulo desiderato.
2. Clicchi su modifica sul widget modulo.
3. Clicchi per aggiungere un nuovo campo e selezioni `Prosopo Procaptcha` dall'elenco.
4. Salvi la pagina.

**2. Widget login**

> Importante: Prima di utilizzare questa integrazione, si assicuri che il modulo di login sia abilitato nelle impostazioni Procaptcha nella scheda `Core forms`.

Per abilitare la protezione per il [widget login](https://elementor.com/help/login-widget-pro/):

1. Modifichi la pagina target con il widget login desiderato.
2. Clicchi su modifica sul widget login.
3. Nella sezione `Form Fields`, abiliti la casella di controllo `Prosopo Procaptcha`.
4. Salvi la pagina.
