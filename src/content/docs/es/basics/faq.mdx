### ¿Qué significa `Site Key not Registered`?
Puede encontrar este error cuando configure el widget de Procaptcha en su sitio web. Este error ocurre cuando el sitekey que está utilizando no está registrado en el sistema Prosopo. Para resolver este error, haga clic en el enlace de Activación en el correo electrónico
o <a href="mailto:support@prosopo.io">contacte al equipo de soporte de Prosopo</a>.

### ¿Qué significa `Invalid Site Key`?
Este error ocurre cuando el sitekey que está utilizando no es válido. Para resolver este error, asegúrese de que está utilizando el sitekey correcto, que puede encontrar iniciando sesión en el [portal de Prosopo](https://portal.prosopo.io).

### ¿Qué significa `Unauthorized origin URL`?
Este error ocurre cuando el dominio del sitekey no coincide con el dominio del sitio web donde se está utilizando el widget de Procaptcha. Para resolver este error, asegúrese de que el dominio del sitekey coincida con el dominio del sitio web iniciando sesión en el [portal de Prosopo](https://portal.prosopo.io) y actualizando el dominio.

### ¿Qué significa `Incorrect CAPTCHA type`?
Este error ocurre cuando el [tipo de captcha](/es/basics/captcha-types) no está configurado correctamente. El `captchaType` predeterminado es `frictionless`. Si desea usar `pow` o `image` debe configurar su `captchaType` en la configuración del widget a `pow` o `image`. También debe actualizar la [configuración del portal](https://portal.prosopo.io) para que coincida.

### ¿Qué significa `Session not found`?
Este error ocurre cuando no se encuentra la sesión. Esto puede suceder si la sesión ha expirado o si el ID de sesión no es válido. Para resolver este error, espere a que se recargue el widget (3s) o actualice la página. Si el error persiste, intente borrar la caché y las cookies de su navegador. Si el error aún persiste, por favor [contacte al equipo de soporte de Prosopo](https://discord.com/invite/3nMYAHecZT).

### ¿Qué significa `Error parsing captcha`?
Este error ocurrirá si se han enviado datos incorrectos con la respuesta del captcha. Para resolver este error, asegúrese de que los datos enviados con la respuesta del captcha sean correctos.
