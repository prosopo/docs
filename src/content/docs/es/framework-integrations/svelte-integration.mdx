Procaptcha proporciona un componente oficial de Svelte para una fácil integración de Prosopo Procaptcha en cualquier proyecto Svelte.
El componente de integración admite todas las [opciones de renderizado de Procaptcha](/es/basics/client-side-rendering/#procaptcha-options).

## 1. Instalación

`npm install @prosopo/svelte-procaptcha-wrapper`

## 2. Ejemplo de uso

### 2.1) Configuración básica

```svelte
<script lang="ts">
import {ProcaptchaComponent} from "@prosopo/svelte-procaptcha-wrapper";
</script>

<ProcaptchaComponent siteKey={"my-site-key"}/>;
```

### 2.2) Uso avanzado

```svelte
<script lang="ts">
import {ProcaptchaComponent} from "@prosopo/svelte-procaptcha-wrapper";

const siteKey = "my-site-key";
const handleVerification = (token: string): void => {
    console.log('verified', token);
};
</script>

<ProcaptchaComponent siteKey="{siteKey}"
language="en"
callback={handleVerification}
htmlAttributes={{class: "my-app__procaptcha"}} />
```
