title: 'User Registration'
description: 'Soporte de User Registration.'
---

El plugin Procaptcha extiende [User Registration & Membership](https://wordpress.org/plugins/user-registration/):

1. Registrando el campo `Prosopo Procaptcha`, que puede agregar fácilmente a sus formularios de registro para la protección de Procaptcha.

Para habilitar esto, abra el formulario de destino e inserte el campo `Prosopo Procaptcha` en cualquier lugar del formulario, encima del botón `Submit`.

2. Extendiendo los formularios personalizados de inicio de sesión y restablecimiento de contraseña.

El plugin User Registration permite no solo crear formularios de registro, sino también crear formularios de inicio de sesión y restablecimiento de contraseña.
Para habilitar su protección, visite la configuración de Procaptcha, abra la pestaña `Formularios principales` y active los interruptores relacionados.
