---

El plugin Procaptcha admite los siguientes formularios de [bbPress](https://wordpress.org/plugins/bbpress/):

**1. Formularios de foro**

El plugin Procaptcha agrega la configuración `Prosopo Procaptcha` a cada foro.

Visite la configuración del foro de destino para habilitar la protección para la creación de temas y formularios de respuesta.

**2. Formularios de cuenta**

bbPress viene con una [serie de shortcodes](https://codex.bbpress.org/features/shortcodes/), que permiten crear formularios de cuenta personalizados: Registro, Inicio de sesión y Contraseña perdida.
Para habilitar su protección, visite la configuración de Procaptcha, abra la pestaña `Formularios principales` y active los interruptores relacionados.
