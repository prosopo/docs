El plugin Procaptcha admite la [función de formularios](https://jetpack.com/forms/) del [plugin JetPack](https://wordpress.org/plugins/jetpack/).

Para agregar Procaptcha a cualquier formulario de Gutenberg de JetPack, siga estos pasos:

1. Agregue el [bloque de grupo](https://wordpress.org/documentation/article/group-block/) dentro del bloque de formulario de destino
2. Agregue el [bloque de shortcode](https://wordpress.org/documentation/article/shortcode-block/) dentro del bloque de grupo agregado
3. Inserte el `[prosopo_procaptcha]` dentro del bloque de shortcode agregado
4. Guarde la página

¡Eso es todo! Ahora puede abrir la página (como usuario no autorizado) para ver el widget de Procaptcha dentro del formulario.
