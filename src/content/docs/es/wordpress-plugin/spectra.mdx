---

El plugin Procaptcha admite la [función de formularios](https://wpspectra.com/docs/spectra-forms-block/) del [plugin Spectra](https://wordpress.org/plugins/ultimate-addons-for-gutenberg/).

Para agregar Procaptcha a cualquier formulario de Gutenberg de Spectra, siga estos pasos:

1. Seleccione el bloque de formulario deseado en el editor de Gutenberg
2. Agregue un campo de entrada oculto: haga clic en el icono más y opte por el campo 'oculto'
3. Use `prosopo_procaptcha` como nombre y valor del campo oculto
4. Guarde la página

¡Eso es todo! Ahora puede abrir la página (como usuario no autorizado) para ver el widget de Procaptcha en el lugar del campo oculto.
