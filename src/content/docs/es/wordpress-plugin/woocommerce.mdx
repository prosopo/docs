---
title: 'WooCommerce'
description: 'Soporte de WooCommerce.'
---

El plugin Procaptcha admite los siguientes formularios de [WooCommerce](https://wordpress.org/plugins/woocommerce/):

**1. Formularios de cuenta personalizados**

La [página de cuenta de WooCommerce](https://woocommerce.com/document/the-my-account-page/) contiene formularios personalizados de Registro, Inicio de sesión y Restablecimiento de contraseña.   
Para habilitar su protección, visite la configuración de Procaptcha, abra la pestaña `Formularios principales` y active los interruptores relacionados.

**2. Formularios de pago**

Hoy en día, WooCommerce incluye 2 tipos de pago diferentes: el clásico y el de bloques. La página de pago utiliza uno de ellos dependiendo de su tema.
El plugin Procaptcha admite ambos; para habilitar la protección visite la configuración de Procaptcha, abra la pestaña `WooCommerce` y active el interruptor relacionado.

**3. Formulario de seguimiento de pedidos**

WooCommerce viene con el [shortcode de formulario de seguimiento de pedidos](https://woocommerce.com/document/woocommerce-shortcodes/) incorporado, que puede agregar a cualquier página.
Para habilitar la protección visite la configuración de Procaptcha, abra la pestaña `WooCommerce` y active el interruptor relacionado.
